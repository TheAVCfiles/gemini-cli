Purpose
-------
This folder is your internal command center for building, shipping, and protecting the Studio Dashboard MVP. Use it to onboard hires, run sprints, and create investor-grade artifacts.

How to use
----------
1. Read `02_Product_Spec_MVP.md` and `03_Roadmap_30-60-90.md`.
2. Use `05_Technical_Specs/devops_README.md` to set up the CI & cosign flow.
3. Onboard with `08_SOPs/onboarding_checklist.md`.

IP & Security
-------------
- Default: local-first, encrypted storage for private assets.
- Exports: client-side signed, encrypted blobs (.enc).
- Sharing: always encrypt exports with `gpg -c` before emailing.
