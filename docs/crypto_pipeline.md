# Cunning Mercy Crypto Pipeline

This optional utility packages the DOGE/ETH technical signal engine with an Elon
meme-pressure overlay. It ships offline-safe defaults so it can run in locked
down environments and optionally switches to live data when network access and
API keys are available.

## Layout

```
scripts/crypto_pipeline/
  pipeline.py          # CLI entry point to generate combined output
  signals/
    __init__.py
    crypto.py          # RSI/MACD/EMA logic for DOGE & ETH
    elon.py            # Meme-pressure scoring for recent Elon tweets
outputs/
  cunning_mercy_crypto.json  # Sample payload generated by the pipeline
```

## Usage

Offline stub mode (no network calls, uses synthetic price series and canned
tweets):

```bash
python scripts/crypto_pipeline/pipeline.py
```

Live mode (requires network access, yfinance installed, and a Twitter v2 bearer
token):

```bash
python scripts/crypto_pipeline/pipeline.py --live --bearer-token "$TWITTER_BEARER_TOKEN"
```

The CLI writes a JSON payload to `outputs/cunning_mercy_crypto.json` and prints
it to stdout. The payload mirrors the theatrical phrasing from the operator
brief and exposes the raw indicator state for downstream automation.
