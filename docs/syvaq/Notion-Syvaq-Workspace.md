# SYVAQ — DeCrypt the Future Workspace

## Client Intake (Page)

**Fields**

- Client Name
- Founder Birthdate & Time (Timezone)
- Founder Birthplace (City, Country)
- Entity Founding Date & Time (Timezone)
- Entity Founding Location (City)
- Contact Info
- Consent: Case Study (Yes/No)
- Guardian Roster (Names + Contact)
- Notes

## Delivery Project (Database)

**Columns**

- Week (Number)
- Dates (Date Range)
- Objective (Text)
- Tasks (Checklist)
- Astro Window (Text)
- Safety Check (Multi-select: Capacity OK / Guardian Ready / Legal Signed)
- Trigger Rule (Text)
- Contingency (Text)
- Owner (Person)
- Completed (Checkbox)

## Astro Engine (Database)

**Columns**

- Date
- Founder Score
- Entity Score
- Synastry Activation
- Composite Score
- Tag (Select: green / amber / red)
- Notes
- Event (Relation → Delivery Project)

## Guardian Protocol (Page)

**Sections**

- Guardian Role & Script
- Aura Mode Flow (low-visibility signal)
- Escalation Matrix
- Consent Script
- Debrief Template

## Case Studies (Page)

**Template**

- Date
- Short Summary (three sentences)
- Signals Leading Up to Event (list)
- Outcome (tangible metrics)
- Consent Flags
