# README Header Kit

## Universal Header

```md
# <Project Name>

**<One-line thesis>.**

<2-3 sentence value proposition that explains what this repo does and why it matters>.
```

## Evidence Locker Header

```md
## Evidence Locker

- Canonical spec: `<path>`
- Verification tooling: `<path>`
- Example bundle(s): `<path>`
- Change log / versioning: `<path>`
```

## Maintainer Blocks

```md
## Maintainers

- Primary maintainer: <name>
- Security contact: <contact>
- Governance owner: <name/team>
```

## Non-negotiables Block

```md
## Non-negotiables

1. Refuse weak provenance.
2. No silent rewrites; version all schema changes.
3. Privacy by default.
4. Deterministic verification outputs.
```

## Status Block

```md
## Status

- Spec: ✅
- v0 implementation: ⏳
- Next ship: <Issue link or ID>
```
